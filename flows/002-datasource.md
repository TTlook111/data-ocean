# 002 数据源管理 — 页面交互简报

---

## 页面 1：数据源管理 `/admin/datasources`

### 布局
顶部筛选工具栏 + 数据表格 + 底部分页器。右上角"新建数据源"按钮。

### 筛选工具栏
| 筛选项 | 类型 | 行为 |
|--------|------|------|
| 数据源名称 | text input | 模糊搜索，防抖 300ms |
| 状态 | select | ENABLED/DISABLED，支持"全部" |
| 健康状态 | select | HEALTHY/UNHEALTHY/UNKNOWN，支持"全部" |
| 重置按钮 | button | 清空筛选，重新加载 |

### 表格列
| 列 | 宽度 | 特殊渲染 |
|----|------|----------|
| 名称 | 150px | 加粗，下方小字显示描述（单行截断） |
| 连接信息 | 200px | 格式：`host:port / dbName`，下方小字显示用户名 |
| 类型 | 80px | Tag："MySQL"（MVP 仅支持 MySQL） |
| 状态 | 90px | Tag：启用=绿，禁用=灰 |
| 健康 | 90px | 图标+文字：✓健康=绿，✗异常=红，?未知=灰 |
| 上次检测 | 140px | 相对时间，hover 完整时间。未检测过显示"—" |
| 授权用户数 | 80px | 数字，点击可打开授权弹窗 |
| 创建者 | 100px | — |
| 操作 | 220px | 按钮组 |

### 操作按钮逻辑
| 按钮 | 条件 | 确认 | 成功反馈 |
|------|------|------|----------|
| 编辑 | 始终 | 无 | 打开编辑弹窗 |
| 测试连接 | 始终 | 无 | 成功=绿色 Message，失败=红色 Message+错误详情 |
| 启用 | 状态=DISABLED | 无 | 行内状态即时更新 |
| 禁用 | 状态=ENABLED | 确认弹窗（含影响提示） | 行内状态即时更新 |
| 授权 | 始终 | 无 | 打开授权弹窗 |
| 删除 | 状态=DISABLED | 确认弹窗（红色） | 行消失 |

### 新建/编辑弹窗

**分步表单（2步）**:

Step 1 — 基本信息：
| 字段 | 校验 | 说明 |
|------|------|------|
| 数据源名称 | 必填，2-100字符，唯一 | — |
| 描述 | 选填，最多 500 字符 | — |

Step 2 — 连接配置：
| 字段 | 校验 | 说明 |
|------|------|------|
| 主机地址 | 必填，IP或域名格式 | — |
| 端口 | 必填，1-65535 | 默认 3306 |
| 数据库名 | 必填 | — |
| 字符集 | select | 默认 utf8mb4 |
| 用户名 | 必填 | 建议只读账号 |
| 密码 | 必填（新建）/选填（编辑） | 编辑时显示"留空则不修改" |

**弹窗底部操作区**:
- "测试连接"按钮（左侧）：点击后显示 loading → 成功/失败反馈
- "取消"按钮
- "保存"按钮：仅在测试连接成功后可点击（首次新建时强制要求）

### 授权管理弹窗

**布局**: 上方为添加区，下方为已授权列表。

添加区：
- 用户搜索下拉（支持按用户名/姓名模糊搜索）
- 支持多选
- "添加"按钮

已授权列表表格：
| 列 | 内容 |
|----|------|
| 用户名 | — |
| 姓名 | — |
| 部门 | — |
| 授权时间 | — |
| 操作 | "取消授权"按钮 |

### 状态矩阵

| 状态 | 表现 |
|------|------|
| 加载中 | 表格 skeleton（5行） |
| 空数据 | 空状态插图 + "暂无数据源，点击右上角按钮添加" |
| 筛选无结果 | "未找到匹配的数据源" + 重置按钮 |
| 测试连接中 | 按钮变为 loading 状态，其他操作不阻塞 |
| 测试成功 | 绿色 Message："连接成功（耗时 XXms）" |
| 测试失败 | 红色 Message + 展开错误详情（连接超时/认证失败/库不存在等） |
| 禁用确认 | "确定禁用数据源 {name}？当前有 N 个用户被授权，禁用后将无法查询" |
| 删除确认 | "确定删除数据源 {name}？此操作不可恢复，所有授权关系将被清除" |
| 删除限制 | 启用状态的数据源不可删除，删除按钮 disabled + tooltip 提示 |

### 交互细节
- 新建时必须先测试连接成功才能保存（按钮灰色 + tooltip 提示）
- 编辑时如果修改了连接参数，保存按钮旁提示"建议重新测试连接"
- 测试连接是异步操作，不阻塞其他交互
- 健康状态由后台定时任务每 5 分钟更新，前端不主动轮询
- 密码字段编辑时显示为空（不回显），placeholder 提示"留空则不修改"
- 分页：默认每页 20 条

### 上下游
- 来源：侧边栏"数据源管理" / 管理首页快捷入口
- 去向：无直接跳转（授权弹窗在当前页内完成）

## 页面 2：问答端数据源选择 `/query`

### 布局
页面标题 + 数据源卡片网格（响应式：大屏3列，中屏2列，小屏1列）+ 底部查询入口区。

### 数据源卡片
```
┌─────────────────────────────┐
│  🗄️  数据源名称              │
│  库名: database_name         │
│  描述文本（最多2行截断）...   │
│                              │
│  [选择此数据源]              │
└─────────────────────────────┘
```

卡片交互：
- hover 时轻微上浮 + 阴影加深
- 点击整个卡片等同于点击"选择此数据源"按钮
- 选中状态：边框高亮 + 左上角勾选标记

### 底部查询入口区
```
┌──────────────────────────────────────────────┐
│  💬 输入你的问题...              [开始查询]   │
└──────────────────────────────────────────────┘
```
- 未选择数据源时：输入框 disabled，placeholder "请先选择一个数据源"
- 已选择数据源时：输入框可用，placeholder "例如：查询上个月销售额最高的10个产品"
- "开始查询"按钮在输入框为空时 disabled

### 状态矩阵

| 状态 | 表现 |
|------|------|
| 加载中 | 卡片区域显示 3 个 skeleton 卡片 |
| 有数据源 | 卡片网格正常展示 |
| 无授权数据源 | 居中空状态：插图 + "暂无可用数据源" + "请联系管理员开通数据源访问权限" |
| 单个数据源 | 自动选中，跳过选择步骤 |
| 数据源加载失败 | 错误状态 + 重试按钮 |

### 交互细节
- 如果用户只有 1 个授权数据源，自动选中并聚焦输入框
- 选择数据源后，输入框自动获得焦点
- 切换数据源时清空输入框内容
- 卡片上不显示连接信息（host/port/用户名），只显示业务信息（名称/库名/描述）
- 页面不显示侧边栏的管理菜单（普通用户视角）

### 上下游
- 来源：普通用户登录后默认页 / 侧边栏"智能查询"
- 去向：选择数据源 + 输入问题 → 查询对话页（008模块，后续开发）

## 跨页面一致性要求

### 表格页通用规范
- 所有管理表格页保持统一的筛选栏 + 表格 + 分页器三段式布局
- 筛选栏高度固定，表格区域自适应剩余高度（flex: 1）
- 分页器固定在底部，显示总条数 + 页码 + 每页条数选择
- 表格无数据时高度不塌缩，保持最小高度 300px
- 所有确认弹窗使用 Element Plus MessageBox，危险操作用红色主题

### 加载态规范
- 首次加载：skeleton 占位（模拟真实布局）
- 刷新/翻页：表格上方显示细长 loading bar，表格内容保持上一次数据（不清空）
- 操作中：仅操作按钮 loading，不阻塞页面其他交互

### 反馈规范
- 成功操作：顶部 Message 提示，2s 自动消失
- 失败操作：顶部 Message 错误提示，需手动关闭
- 确认弹窗：标题明确动作，内容说明影响，危险操作按钮用红色
- 表单校验：字段下方红色文字，blur 时触发单字段校验，submit 时触发全量校验
